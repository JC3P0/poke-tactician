### Why Gen 1?

Generation I mechanics are unique, quirky, and historically significant:
- **DVs (0-15)** instead of modern IVs (0-31)
- **Unified Special stat** (no Sp.Atk/Sp.Def split)
- **Speed-based critical hits** (not random!)
- **Type-based Physical/Special** (determined by move type)
- **The Ghost vs Psychic bug** (Ghost deals Ã—0 to Psychic!)
- **15 types** (no Dark, Steel, or Fairy)

This project implements these mechanics with academic rigor and proper citations.
